Here are instructions for the files in this folder.

gui.py
By running this file, you can simulate the crowd evacuation on the GUI. Import "finlandia_talo.py" if you want to simulate the evacuation from the conference building, or use "simple_scenario.py" if you want to simulate the evacuation from the hexagon-shaped area.

finlandia_talo.py
Contains the floor of the conference building.

simple_scenario.py
Contains the floor of the hexagon-shaped area.

agents_initialization_conference.npy / agents_initialization_simple.npy
Data file that includes the initial positions of the exiting agents. We need this so that the guides are not positioned so that they overlap with the exiting agents.
